# Sistema de Lista de Tarefas

Este trabalho consiste no desenvolvimento de um sistema simples de gestão de tarefas em Python. O objetivo do programa é permitir ao utilizador criar, organizar e acompanhar tarefas através de um menu interativo no terminal.

O sistema utiliza conceitos fundamentais de programação, como classes, objetos, listas, funções e laços de repetição. Cada tarefa possui uma descrição e um estado, podendo estar pendente ou concluída.

Entre as principais funcionalidades do programa estão:

* Adicionar novas tarefas;
* Listar todas as as tarefas;
* Marcar tarefas como concluídas;
* Visualizar apenas tarefas pendentes;
* Visualizar apenas tarefas concluídas;
* Consultar os detalhes de uma tarefa específica.

Este projeto foi desenvolvido com o objetivo de aplicar conhecimentos de programação orientada a objetos e estruturas de dados em Python, promovendo também a organização e gestão de tarefas de forma simples e prática.
